# Mobile App Downloads

Put the final Android APK here with this exact filename:

```text
cricket-management-pro.apk
```

The website download button points to:

```text
downloads/cricket-management-pro.apk
```

To create an APK from the Expo app:

```bash
cd mobile-app
npm run build:android:apk
```

After the build finishes, download the generated APK from EAS and place it in this folder.
