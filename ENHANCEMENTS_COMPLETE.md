## How to Use

### Opening the Application
1. Open `index.html` in your web browser
2. The application loads with the Dashboard view
3. All data is stored locally in your browser

### Key Features
- **Dashboard** - Overview and quick stats
- **Tournaments** - Create and manage tournaments
- **Live Scorer** - Score matches in real-time with 3D field
- **Players Directory** - Manage player profiles
- **Match History** - Review past matches

### 3D Field Visualization
1. Click "Live Scorer" in the sidebar
2. Start a match
3. Click the "3D Field" button in the live stream section
4. Use Rotate Left/Right and Zoom In/Out buttons to interact
