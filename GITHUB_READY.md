## 🎯 Quick Start for GitHub

### Step 1: Create GitHub Repository
1. Go to https://github.com/new
2. Name it: `Cricket-Management-Pro`
3. Make it Public
4. Click "Create repository"

### Step 2: Push to GitHub
```bash
# Navigate to this folder
cd "Cricket-Management-Pro"

# Initialize git
git init

# Add all files
git add .

# Create commit
git commit -m "Initial commit: Cricket Management Pro v1.0.0"

# Add remote (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/Cricket-Management-Pro.git

# Push to GitHub
git branch -M main
git push -u origin main
```

### Step 3: Enable GitHub Pages (Optional)
1. Go to Settings → Pages
2. Select "main" branch
3. Your app will be live at: `https://YOUR_USERNAME.github.io/Cricket-Management-Pro/`
